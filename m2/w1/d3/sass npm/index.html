grazie a npm install si possono installare dei pacchetti

questi pacchetti si possono aggiornare automaticamente con npm update

npm install -g sass è già stato fatto e non bisogna rifarlo

fare npm init per ogni nuovo progetto per utilizzare il pacchetto sass

fare npm install node-sass dopo npm init per creare i file necessari per lavorare in sass

fare npm run + nome comando per far sì che i due file scss e css si sincronizzino. Dopodiché si potrà editare solo il file scss
